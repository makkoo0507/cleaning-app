# ユースケース

**プロジェクト名:** 民泊清掃管理アプリ  
**作成日:** 2026-06-20  
**バージョン:** 1.0

---

## アクター一覧

| アクター | 利用画面 |
| --- | --- |
| プラットフォーム運営者 | 直接 DB 操作 または 専用管理画面 |
| 業者（管理者） | 管理 Web アプリ |
| 業者（社員） | 管理 Web アプリ |
| 清掃者 | LINE Mini App |
| 物件関係者 | LINE Mini App |

---

## UC-01｜プラットフォーム運営者

| UC番号 | ユースケース名 | 概要 |
| --- | --- | --- |
| UC-01-01 | テナント登録 | 業者を新規テナントとして登録する |
| UC-01-02 | プラン変更 | テナントのプランをフリーから有料に変更する |
| UC-01-03 | テナント無効化 | 利用停止したテナントを無効化する |

---

## UC-02｜業者（管理者）

| UC番号 | ユースケース名 | 概要 |
| --- | --- | --- |
| UC-02-01 | 社員登録 | 社員アカウントを作成しログイン情報を発行する |
| UC-02-02 | 社員編集・削除 | 社員情報の変更またはアカウントを削除する |
| UC-02-03 | 物件登録 | 民泊物件を登録する |
| UC-02-04 | 物件編集・削除 | 物件情報を変更または削除する |
| UC-02-05 | 物件関係者登録・招待 URL 発行 | オーナー・運用担当・営業などを登録し LINE 紐付け用の招待 URL を発行する |
| UC-02-06 | 物件関係者の通知設定変更 | 物件ごとに関係者の LINE 通知 ON/OFF を変更する |
| UC-02-07 | 清掃者登録・招待 URL 発行 | フリーランス清掃者を登録し LINE 紐付け用の招待 URL を発行する |
| UC-02-08 | 清掃者編集・削除 | 清掃者情報を変更または削除する |
| UC-02-09 | スケジュール作成 | 清掃案件を作成し清掃者をアサインする |
| UC-02-10 | スケジュール編集・削除 | 清掃案件の日時・担当者を変更または削除する |
| UC-02-11 | 清掃記録閲覧 | 全案件の清掃記録を一覧・詳細表示する |
| UC-02-12 | 請求・支払い金額記録 | 案件ごとの請求額・支払い額を記録する |
| UC-02-13 | 月次集計・CSV 出力 | 期間別に請求・支払いを集計し CSV をエクスポートする |

---

## UC-03｜業者（社員）

| UC番号 | ユースケース名 | 概要 |
| --- | --- | --- |
| UC-03-01 | スケジュール作成 | 清掃案件を作成し清掃者をアサインする |
| UC-03-02 | スケジュール編集 | 清掃案件の日時・担当者を変更する |
| UC-03-03 | 清掃記録閲覧 | 全案件の清掃記録を閲覧する |
| UC-03-04 | 請求・支払い金額閲覧 | 案件ごとの請求額・支払い額を閲覧する（編集不可） |

---

## UC-04｜清掃者

| UC番号 | ユースケース名 | 概要 |
| --- | --- | --- |
| UC-04-00 | LINE アカウント紐付け | 業者から送られた招待 URL をタップし LINE ログインでアカウントを紐づける |
| UC-04-01 | マイスケジュール確認 | 自分がアサインされた案件一覧を確認する |
| UC-04-02 | 案件詳細確認 | 物件住所・清掃日時・特記事項を確認する |
| UC-04-03 | 清掃開始記録 | 清掃開始時刻を登録する |
| UC-04-04 | 清掃完了記録 | 清掃完了時刻・メモを登録する |

---

## UC-05｜物件関係者

| UC番号 | ユースケース名 | 概要 |
| --- | --- | --- |
| UC-05-00 | LINE アカウント紐付け | 業者から送られた招待 URL をタップし LINE ログインでアカウントを紐づける |
| UC-05-01 | スケジュール確認 | 自物件の清掃予定一覧を確認する |
| UC-05-02 | 清掃記録閲覧 | 完了した清掃の記録（完了時刻・所要時間・メモ）を閲覧する |
