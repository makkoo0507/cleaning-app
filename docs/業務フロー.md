## BF-01｜初期セットアップ

業者が初めてアプリを使い始める際のセットアップ手順。LINE 紐付けの詳細は BF-02 を参照。

```
業者管理者
  │
  ├─ 1. 物件を登録
  │       └─ 物件名・住所・特記事項を入力
  │
  ├─ 2. 物件関係者を登録
  │       ├─ ユーザー情報（名前）を登録 → invite_token 発行
  │       ├─ 物件への紐付け・役割（owner / operations / sales）・通知設定
  │       └─ 招待 URL を LINE や SMS で関係者本人に送付
  │
  ├─ 3. 清掃者を登録
  │       ├─ ユーザー情報（名前）を登録 → invite_token 発行
  │       └─ 招待 URL を LINE や SMS で清掃者本人に送付
  │
  └─ 4. 社員を登録（必要な場合）
          └─ メール・パスワードでアカウント発行
```

## BF-02｜LINE アカウント紐付け

物件関係者・清掃者が初回利用時に LINE アカウントを紐づける。

```
業者管理者
  └─ 1. 管理 Web で招待 URL を発行・コピー
          └─ URL 例: https://example.vercel.app/liff/invite?token={invite_token}

  └─ 2. 招待 URL を LINE または SMS で相手に送付（アプリ外）

対象者（清掃者 または 物件関係者）
  └─ 3. 招待 URL をタップ → LINE Mini App が起動

  └─ 4. LINE でログイン（LIFF 認証）

  └─ 5. システムがトークンを照合
          ├─ 一致: users.line_user_id を保存 → invite_token を NULL クリア
          └─ 不一致 / 期限切れ: エラー表示

  └─ 6. 紐付け完了 → LINE 通知が届くようになる
```

## BF-03｜スケジュール登録

チェックアウト日が確定したタイミングで清掃案件を登録する。

```
業者（管理者 または 社員）
  │
  ├─ 1. スケジュール作成
  │       ├─ 対象物件・清掃日・開始予定時刻を入力
  │       └─ 清掃者をアサイン（後でアサインも可）
  │
  └─ 2. 作成完了 → 自動通知
          │
          ├─ 清掃者へ LINE 通知
          │       「〇月〇日 〇〇物件の清掃が入りました」+ 詳細リンク
          │
          └─ 物件の通知有効な関係者へ LINE 通知
                  「〇月〇日に清掃が予定されました」+ 詳細リンク
```

## BF-04｜清掃実施

清掃当日の記録登録から完了通知までの流れ。

```
【前日】
  システム（自動）
    └─ 清掃者へ LINE リマインド通知
            「明日 〇〇物件の清掃があります」

【当日】
  清掃者（LINE Mini App）
    │
    ├─ 1. マイスケジュールで案件・物件情報を確認
    │
    ├─ 2. 清掃開始 → 開始時刻を登録
    │
    ├─ 3. 清掃作業
    │
    └─ 4. 清掃完了 → 完了時刻・メモを登録
            │
            └─ 完了 → 自動通知
                    │
                    ├─ 物件の通知有効な関係者へ LINE 通知
                    │       「清掃が完了しました（完了時刻・所要時間）」
                    │
                    └─ 業者へ LINE 通知
                            「〇〇さんが〇〇物件の清掃を完了しました」

【完了後】
  物件関係者（LINE Mini App）
    └─ 清掃記録（完了時刻・所要時間・メモ）を閲覧

  業者（管理 Web）
    └─ 清掃記録を一覧で確認
```

## BF-05｜請求・支払い管理

月次または任意のタイミングで請求・支払いを管理する。

```
業者管理者（管理 Web）
  │
  ├─ 1. 案件ごとに請求額・支払い額を記録
  │       ├─ 民泊オーナーへの請求額
  │       └─ 清掃者への支払い額
  │
  ├─ 2. 月次集計を確認
  │       └─ 期間を指定して合計を表示
  │
  └─ 3. CSV エクスポート
          └─ 経理処理・支払い手続きを手動で実施
```
